## Setup

Create a virtual environment

```bash
$ python -m venv .venv
$ source .venv/bin/activate
```

Install [pytest](https://pypi.org/project/pytest/)
```bash
(.venv) $ pip install pytest
```

## Run the tests with pytest

```bash
(.venv) $ make test
```
